LINE連携｜3つの値の取得手順

お店のLINE設定画面から3つの値をコピーし、設定画面の同じ番号の欄に貼り付けます。所要時間はおよそ10分です。すべて developers.line.biz/console の中だけで完結します。

はじめに（言葉の意味）
  • プロバイダー＝お店専用の「箱」。この箱の中に、下の2つの機能を作ります。
  • Messaging APIチャネル＝お客様へお知らせを送る機能。①と②はここで取ります。
  • LINEログインチャネル＝お客様が連携する画面の機能。③はここで取ります（後で新しく作ります）。

大事なルール：①②③は必ず同じ1つのプロバイダー（箱）の中に作ります。別の箱に作ると連携できません。

値 どこで取る？
① アクセストークンMessaging APIチャネル →「Messaging API設定」タブ
② Channel secret同じMessaging APIチャネル →「チャネル基本設定」タブ
③ LIFF ID新しく作る「LINEログイン」チャネル →「LIFF」タブ
1① アクセストークンを取る

📍今いる場所：Messaging APIチャネル（お知らせ送信用）

  1. developers.line.biz/console を開き、お店のアカウントでログイン → お店のプロバイダー（箱）→ Messaging APIのチャネルを開きます。
  2. 上部のタブで「Messaging API設定」を選びます。
  3. 下へスクロールし「チャネルアクセストークン（長期）」の〔発行〕を押します。
  4. 表示された長い文字列を〔コピー〕→ 設定画面の①の欄へ貼り付け。
2② Channel secret を取る

📍今いる場所：同じMessaging APIチャネル（移動しなくてOK）

  1. 同じチャネルの上部タブで「チャネル基本設定」を選びます。
  2. 少し下にある「Channel secret」の値を〔コピー〕→ 設定画面の②の欄へ貼り付け。
3③ LIFF ID を取る

📍今いる場所：新しく作る「LINEログイン」チャネル

③だけは別の機能です。LIFFはMessaging APIチャネルには作れないので、同じプロバイダー（箱）の中に「LINEログイン」チャネルをもう1つ作ってから追加します。

  1. developers.line.biz/console でお店のプロバイダー（①②と同じ箱）を開きます。
  2. 「新規チャネル作成」→ 種類で「LINEログイン」を選びます（「LINEミニアプリ」ではありません）。
  3. 入力して作成：
    • チャネル名：例「お店名 連携」（お客様のログイン画面に表示されます）
    • アプリタイプ：「ウェブアプリ」にチェック
    • その他の必須項目を入力
  4. 作成したLINEログインのチャネルを開き、上部の「LINEログイン設定」タブを選びます。「コールバックURL」の欄に https://benriyamanzoku-maker.github.io/logiqa-liff/ を登録して〔更新〕します。これが無いと、連携時に「400 Bad Request」エラーになります（必須）。
  5. 同じチャネルで上部の「LIFF」タブを開きます。「登録されているLIFFアプリはありません。」と表示されたら、緑色の〔追加〕ボタンを押します。
  6. 入力：
    • LIFFアプリ名：車屋さんのお知らせ便（このサービス名。連携画面にも表示されます）
    • サイズ：Full
    • エンドポイントURL：https://benriyamanzoku-maker.github.io/logiqa-liff/
    • スコープ：profile と openid にチェック
    • 友だち追加オプション：On（normal）（連携と同時にお店の友だち追加を促せます）
    • Scan QR：オフのまま
    • モジュールモード：オフのまま
  7. 〔追加〕すると「LIFF ID」（例：2010315487-xxxxxxxx）が表示されます。〔コピー〕→ 設定画面の③の欄へ貼り付け。
①アクセストークンと②Channel secretは「銀行の暗証番号」と同じ大切な鍵です。第三者に教えたり、メールやSNSに貼ったりしないでください。この画面で保存したあとは、安全のため再表示されません。
設定が難しい場合は、すべて株式会社LOGIQA（info@logiqa-inc.com）が代行します。お気軽にご連絡ください。
